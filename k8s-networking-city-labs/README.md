# Kubernetes Networking City Labs

Welcome to the Kubernetes Networking City Labs curriculum. This repository provides hands-on labs using a city analogy to teach Linux and Kubernetes networking concepts.

## Structure
- **linux-labs/**: Foundational networking using Linux tools
- **kubernetes-equivalents/**: Mapping Linux concepts to Kubernetes
- **failure-labs/**: Common failure scenarios and troubleshooting
- **guides/**: Glossary, troubleshooting, and workshop materials
- **assets/**: Visuals and diagrams

## Getting Started
Run `bootstrap/bootstrap.sh` to generate the full lab structure from templates and data.
