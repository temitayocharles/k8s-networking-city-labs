## Scenario Description
In this lab, the "Front Desk" represents the service abstraction in Kubernetes networking. You will use iptables to simulate service traffic redirection, demonstrating how services act as traffic indirection points rather than endpoints, similar to a front desk routing visitors to the correct apartment.
